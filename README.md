# Vision-Based Gait Abnormality Detection Using Pose Estimation

## Overview

This project detects gait abnormalities from walking videos using computer vision and machine learning techniques.

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- Pandas
- Scikit-Learn

## Features

- Pose Landmark Extraction
- Feature Engineering
- Data Validation
- Machine Learning Model Training
- Gait Abnormality Prediction

## Workflow

Video Input
↓

↓

↓
Data Validation
↓

↓
Prediction
